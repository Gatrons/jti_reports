## ðŸš€ Tahapan Penggunaan Aplikasi

### 1. Login / Registrasi
- Buka aplikasi **JTI Report**
- Login menggunakan **Email & Password** atau **Google**
- Jika belum memiliki akun, pilih **Daftar**
- Pastikan email telah **diverifikasi** sebelum login

---

### 2. Halaman Beranda
Setelah login berhasil, pengguna akan masuk ke halaman utama yang berisi:
- Menu tambah laporan
- Riwayat laporan
- Notifikasi
- Profil pengguna

---

### 3. Membuat Laporan Kerusakan
1. Pilih menu **Tambah Laporan**
2. Isi data laporan:
   - Jenis kerusakan
   - Lokasi fasilitas
   - Deskripsi kerusakan
   - Tingkat keparahan
3. Unggah **foto atau video pendukung** (maksimal 3 media)
4. Tekan tombol **Kirim Laporan**
5. Laporan akan tersimpan dan dikirim ke admin

---

### 4. Melihat Riwayat & Status Laporan
- Buka menu **Riwayat Laporan**
- Pengguna dapat melihat:
  - Daftar laporan yang pernah dibuat
  - Status laporan (Diajukan, Diproses, Selesai)
  - Detail laporan
  - User dapat mengubah/menghapus laporan jika salah input dengan update melalui detail laporan

---

### 5. Notifikasi
- Pengguna menerima notifikasi saat:
  - Laporan diproses oleh admin
  - Status laporan diperbarui
- Notifikasi dapat dibuka melalui **ikon lonceng**

---

### 6. Profil
Pada menu **Profil**, pengguna dapat:
- Melihat informasi akun
- Mengubah password
- Logout dari aplikasi

---

### 7. Akses Admin (Khusus Admin)
Admin memiliki akses khusus untuk:
- Melihat seluruh laporan pengguna
- Mencari dan memfilter laporan
- Mengubah status laporan
- Memantau laporan secara real-time
